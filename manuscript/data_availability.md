# Data Availability Statement

The data and code that support the findings of this systematic literature review are made fully available to ensure reproducibility:

1. **Repository:** The complete repository, including raw data, processed datasets, and figure generation scripts, is available at:
   `https://github.com/[placeholder-user]/GPS_Denied_Drone_SLR_3.0` (or similar repository link).

2. **Cleaned & Classified Dataset:** The final processed databases (`database_final.csv` and `classified_papers.csv`) can be found in the `data/processed/` directory.

3. **Software & Reproduction Code:** The Python pipeline scripts used for data cleaning (`01_clean_data.py`), taxonomy classification (`02_classify_papers.py`), figures/tables generation (`03_generate_figures.py`), accuracy extraction (`04_extract_accuracy.py`), reference building (`05_build_references.py`), and report compilation (`06_write_report.py`) are fully open-source and documented under `analysis/scripts/`.

4. **Bibliography & References:** The formatted bibliography and BibTeX files are located in the `references/` directory.

All datasets can be deposited on public research repositories such as Zenodo or Figshare under a Creative Commons Attribution license (CC-BY 4.0).
